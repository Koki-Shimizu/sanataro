== Kakeibo Sanataro {<img src="https://secure.travis-ci.org/kaznum/sanataro.png" />}[http://travis-ci.org/kaznum/sanataro]
Sanataro - Simple Web Household Account Book.
built with Ruby on Rails 3.2.

== Functional Features
* Simple kakeibo Web Application.
* Multi-User capable
* Balance adjustment
* Lightweight and beautiful monthly/yearly P/L and B/S charts
* Credit-card due setting
* Simple input User-Interface for iPhone.
* See more on a demo page. http://k.nu-chon.org

== Architectual Features
* All background calcuration is fully tested with rspec.
  * Keeping Controller/Model/Helper C0 test coverage to be 100%.
* Only HTML, CSS, JavaScript. No Flash, No ActiveX.
* Locale Support (Now only Japanese, but you can add your own language easily)

== Requirement
* Ruby: MRI (>= 1.9.2)
* Ruby on Rails: 3.2.0
* DB: 
  * MySQL(adapter mysql2) (tested)
  * SQLite3 (tested)
  * PostgreSQL (heroku available) (not tested)
* Recommend: Passenger and Webserver such as Apache

== If you want to try instantly...
* `git clone git://github.com/kaznum/sanataro.git'
* `cd sanataro'
* `cp config/database.yml.sample config/database.yml'
* Edit config/database.yml if needed
* Create database if you use MySQL or PostgreSQL and set privileges to the database.
* `gem install bundler' if bundler is not installed.
* `bundle install'
* `rake db:migrate'
* `rails s'
* Access http://localhost:3000/ with your favorite web browser.

== Caution
* http(s)://<your URL>/admin/users lists the account names which have been registered, but this page has no authentication logic. Please limit the users by web server or Rails mechanism who can access this page or remove the file "app/controllers/admin/users_controller.rb".
