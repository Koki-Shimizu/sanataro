Welcome to Sanataro

This simple web household account book application is built on Ruby on Rails and supports several DBMS products.

How to try...

$ cp config/database.yml.sample config/database.yml
$ vi config/database.yml
$ bundle install
$ rake db:setup
$ rake db:migrate
$ rails s

DEMO

http://sanataro.heroku.com

  • 3.2.3

Supported Ruby implementations and versions

  • MRI 1.9.2-p250
  • MRI 1.9.3-p125
  • JRuby 1.6.7 (1.9 mode)

Supported DBMS

  • MySQL5
  • PostgreSQL
  • SQLite3
